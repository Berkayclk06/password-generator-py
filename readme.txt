## TO ADD PASSWORD ##

- Enter the website name
- Enter you mail address
- Enter or generate a password
- If you choose to generate a password it will automatically copy it to clipboard
- After pressing add, it will create a data.json file and add your entries to it. If you created .json file before it will append the entry.

## TO SEARCH PASSWORD ##

- Enter the website name
- Pop-up will show email and pass for the website
